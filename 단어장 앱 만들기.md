아래는 우리가 만들 영단어 학습 앱 "호랑이 잉글리시" 기획이야. 

### **단어장 앱 만들기**

앱 이름: 호랑이 잉글리시 Horang English 

대상: 고등학생

단어범위: 구글시트에 암기해야할 영단어 정리한 것
하루에 암기해야할 단어가 그날의 단어장이며 타이틀은 Day1, Day2... 순차적으로 붙인다.
하루에 암기해야할 단어 갯수를 설정에서 정할 수 있으며 구글 시트에 있는 단어를 랜덤으로 갖고 온다.
구글 시트에는 2300개 정도의 단어가 있다. 표제어와 파생어가 단어장에 나와야할 단어이고 쓰기에 동사변형에 대한 언급이 있으면 그것을 단어장에 표현해주고 시험에도 출제한다.
예문은 대학수학능력시험의 스타일로 설정에 있는 난이도로 만들어야한다. 구글시트에 없음. 예문을 로컬 SQLLight에 저장해도 무리가 없을지 검토해주길 바란다. 다른 대안이 있다면 알려줘라.
모든 단어장의 단어는 중복 되지 않는다. 예: Day1의 단어가 Day5나 다른 단어장에 절대 다시 나타타지 않는다.
앱에 접속하면 1개의 단어장이 생성되며 하루에 1개 이상은 생성되지 않는다.

복습 루틴 기준: 학습 날짜에서 -1일, -3일, -7일, -14알, -30일, -60일, -120일 전 학습한 단어장.

나의 물리적 개발환경: 집, i-mac late 2015 monterey 12.7.6 / 학교: windows 11 pc / 핸드폰: 아이폰 12 pro

### **앱의 각 화면들 설명**

- 스플래시
    - 호랑이 이미지와 함께 호랑이 잉글리시 Horang English
- 첫화면: 새로 외워야할 단어장 화면
- 모든 화면 타이틀 좌측에는 호랑이 얼굴 메뉴 버튼이 있어서 그것을 클릭하면 메뉴들이 나온다
    - 단어장리스트
    - 낮가림이 심한 단어
    - 나의 업적
    - 설정
- 단어장 화면 구성
    - 각 단어장의 타이틀은 Day1부터 시작한다.
    - 복습, 테스트 버튼이 있다.
    - 단어장은 학습하는 페이지이다
    - 3개의 컬럼의 테이블 형태로 단어, 뜻, 예문이 나온다.
    - 단어, 뜻, 예문 컬럼의 헤더에는 눈모양 아이콘이 있어서 클릭하면 해당 컬럼의 내용이 모두 가려진다.
        - 성능 이슈 대안: 100개 이상의 셀을 가려야할 수 있으므로 화면에 보이는 15~20개 행만 실제 렌더링/애니메이션
        - stagger(순차 지연): 행마다 10~20ms씩 시차를 두고 애니메이션 시작 → GPU 부하 분산 + 시각적으로 자연스러움
    - 각 셀은 클리하면 가려진다. 가려지는 효과는 유령이미지가 왼쪽으로 빠르게 이동하면서 글자들은 먼지처럼 사라진다.
        - 성능 이슈 대안: 실제로 글자 자체에 파티클 효과를 주는 것이 아니라 먼지가 흩어지는 듯한 애니메이션을 텍스트 위에 얹고 투명도를 주는 방식 고려해줘.
    - 각 영단어를 클릭하면 발음을 들을 수있다. TTS. 예문에는 발음 듣기 기능을 넣지 않는다.
    - 각 단어의 앞에는 아이콘이 있다. 이 아이콘의 목적은 단어를 머리 속에서 인출할 때 기억 나지 않는 정도를 표시하기 위해서이다. 한번 기억이 안 난 것, 두 번 기억이 안 난 것, 세 번 기억이 안 난 것, 네 번 기억이 안 난 것, 다섯 번 기억 안 난 것, 총 5단계이다. 오른쪽으로 swap하면 증가, 왼쪽으로 swap하면 감소한다.
    - 뜻 컬럼에는 모든 주요 뜻이 품사 별로 나오며 예문도 품사 별로 있다. 뜻의 범위와 예문의 수준 및 스타일은 대학수학능력시험 기준이다. 그리고 설정에서 세팅한 수준(고1,고2,고3)을 따른다.
- 단어장 > 복습 화면 구성
    - 학습한 날짜의 -1, -3, -7, -14, -30, -60, -120일 학습한 단어장이 최근 순으로 나온다.
    - 복습화면에서는 뜻과 예문이 가려져있다.
- 단어장 > 테스트 화면 구성
    - 당일 학습한 단어와 복습할 날짜의 단어들이 랜덤으로 뒤섞여 출제된다.
    - 2개의 컬럼: 문제, 답
    - 점수 메기기 버튼을 클릭하면 정답 컬럼과 발음확인 컬럼, 오답 체크, 발음 헷갈림 컬럼이 생성된다. 점수 메기기 버튼은 점수 확인 버튼으로 변경된다.
    - 오답 커럼을 클릭하면 X 표시가 표현된다.
    - 점수 확인 버튼을 클릭하면 점수 확인 애니메이션- 코끼리가 북을 두드리는 애니메니션이 3초 진행되고 점수가 짠 하고 나온다.
    - 애니메니션 화면에 취소 버튼이 있어서 클릭하면 점수 메기기 화면으로 다시 이동한다.
    - 점수 확인 화면에는 나의 업정보기 버튼과 다시 메기기 버튼이 있다.
    - 발음 등록을 어떻게 구현했는지 작업 끝나고 기록에 남겨줘
    - 발음 등록 기능으로 인해 우려되는 사항과 고려해야할 성능 문제를 제기해줘.
- 단어장리스트 화면 구성
    - 검색
    - 낯가림이 심한 단어
    - Day1 부터 모든 단어장의 리스트
- 낯가림이 심한 단어 화면 구성
    - 오답 횟수가 1번 이상인 단어 중 횟수 많은 순으로 상위 10개
    - 단어장 테이블 형식 재사용
- 나의 업적 화면 구성
    - 현재 수준:
    - 이달의 Income
    - 최근 5일 동안의 점수
        - 날짜, 단어장, 점수(100점을 만점으로 환산), Income, 입금여부 체크
        - Income, 입금여부 체크 = 실물 용돈 장부 (실제 결제 연동 X, 그냥 "이 점수면 얼마" Income에 표시하고 부모님이 지급했는지 사용자가 체크 하는 장부 개념)
    - 낯가림이 심한 단어: 상위 10개 단어 나열, 단어만 가로로 나열.
    - 시험 정답 기준 최종 입력 값이 정답 단어의 누적 개수로 추이 그래프
    - 월별 Income 추이
    - 점수 추이
    - 오답 갯수 추이
- 설정 화면 구성
    - 수준 설정: 고1, 고2, 고3
    - 하루동안 외울 단어 개수 설정: 아직 진행하지 않은 단어들로만 단어장을 재생성 한다. 이미 진행한 단어장은 그대로 둔다.
    - 점수별 Income 액수 세팅. 추가, 점수 입력, Income 입력
